# Security Policy

`recon-agent` is for authorized security testing and bug bounty programs.

## Rules

- Test only targets where you have written authorization.
- Stay inside published program scope.
- Respect rate limits and safe harbor rules.
- Do not use this tool for unauthorized scanning or exploitation.

## Reporting Issues

Open a private GitHub security advisory for vulnerabilities in this project. Include reproduction steps, impact, and suggested mitigation if known.
